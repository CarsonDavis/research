# Research

A repository for researching topics with clear guidelines on how it should work.

## Guidelines

- [RESEARCH_GUIDELINES.md](RESEARCH_GUIDELINES.md) - Two-phase workflow: gather notes with citations, then distill into reports
- [PYTHON_GUIDELINES.md](PYTHON_GUIDELINES.md) - Use uv, write modular code, README-driven development
- [GIT_GUIDELINES.md](GIT_GUIDELINES.md) - Add then commit (separately), clean commit messages

## Agents

Custom Claude Code agents live in [agents/](agents/). Copy them to `~/.claude/agents/` to install.
